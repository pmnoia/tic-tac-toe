# Tic-Tac-Toe

A classic, interactive Tic-Tac-Toe game built using React. This project was developed by following the [Official React Tutorial](https://react.dev/learn/tutorial-tic-tac-toe) to explore the fundamentals of components, props, and state management.

## ✨ Current Features

The core gameplay mechanics are fully functional:

- **Interactive Board:** Renders a standard 3x3 Tic-Tac-Toe grid.
- **Turn-Based Play:** Two players can alternate turns playing as "X" and "O".
- **Win Detection:** Automatically calculates and displays the winner when a player gets three in a row.
- **State Protection:** Prevents players from overwriting squares that have already been played.

## 🚧 Work in Progress (Coming Soon)

- **Time Travel:** Implementing an interactive history list that allows players to "jump back" in time to view previous moves and alter the outcome of the game.

## 🛠️ Tech Stack

- **Frontend:** React
- **Styling:** CSS

## 🚀 Getting Started

Follow these instructions to run the game on your local machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/your-username/react-tic-tac-toe.git](https://github.com/your-username/react-tic-tac-toe.git)
   ```

2. **Navigate to the project directory:**
   ```bash
   cd react-tic-tac-toe
   ```
3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Running the Game:**
   npm start
